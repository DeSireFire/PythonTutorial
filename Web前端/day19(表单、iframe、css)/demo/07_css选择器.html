杨满学说，11日零时24分，陕西省森工医院接到西安市120紧急通知，秦岭一号隧道口发生重大交通事故，该院救护车立即出发赶往事故现场。医院启动重大抢救应急预案，骨科、普外科、胸外科、神经外科、麻醉科、急诊科的医护人员和相关辅助科室都做好了接诊救治工作。

目前，该院收治的6名伤员中，4名病情危重，2名严重复合伤。两名伤员正在手术，所有伤员都在救治中，患者生命体征基本平稳，抢救工作在继续进行中。
